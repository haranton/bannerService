# bannerService

**bannerService** — это бэкенд-приложение, реализующее хранение вопросов и ответов с использованием REST API. Проект написан на языке Go и использует PostgreSQL для хранения данных, а также Docker для контейнеризации.

## Возможности

- **CRUD-операции** для вопросов и ответов
- Проверка существования вопроса перед добавлением ответа
- Удаление вопроса с каскадным удалением ответов
- Использование **GORM** для работы с базой данных
- Поддержка миграций с помощью **Goose**
- Логирование с кастомным логгером
- Возможность выбора профиля работы (PostgreSQL или in-memory)

## Технологический стек

- **Язык программирования**: Go 1.21+
- **База данных**: PostgreSQL
- **ORM**: GORM
- **Миграции**: Goose
- **Контейнеризация**: Docker, Docker Compose
- **Логирование**: slog
- **Тестирование**: Testify (есть интеграционный тест)
